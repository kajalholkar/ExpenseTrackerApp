# ExpenseTrackerApp
ExpenseTrackerApp is a simple CRUD-based web application built using ASP.NET Core that helps users manage and track their daily expenses efficiently.

🚀 Features (CRUD Operations)
➕ Create: Add income and expense records
📖 Read: View all transactions and summaries
✏️ Update: Edit existing expense entries
❌ Delete: Remove unwanted records
📊 Category-wise expense tracking (Food, Travel, Bills, etc.)
💰 Real-time balance calculation

🛠️ Tech Stack
ASP.NET Core / MVC
C#
SQL Server
HTML, CSS, JavaScript

🎯 Purpose
This project demonstrates CRUD operations with database integration and helps users manage personal finance effectively.
